## 🚀 초간단 설치 (3분)

### 1단계: 폴더 만들기
```
바탕화면에서:
마우스 오른쪽 클릭 → 새로 만들기 → 폴더
폴더 이름: "naver_map"
```

### 2단계: 파일 복사
```
위의 필수 파일 5개를 폴더에 복사:
✅ naver_map_bulk_downloader.py
✅ requirements.txt
✅ 리스트_네이버지도링크추가.xlsx
✅ 설치.bat
✅ 실행.bat
```

### 3단계: 설치
```
"설치.bat" 더블클릭 → 완료!
```

### 4단계: 실행
```
"실행.bat" 더블클릭 → "2" 입력 → Enter → 완료!
```

## ⚙️ 사전 요구사항

### 1. Python 설치
- **다운로드**: https://www.python.org/downloads/
- **설치 시 주의**: "Add Python to PATH" **반드시 체크**
- **버전**: Python 3.8 이상 권장 (3.11 추천)

### 2. Chrome 설치
- **다운로드**: https://www.google.com/chrome/
- **필수 이유**: Selenium이 Chrome을 사용하여 웹 자동화

### 3. 인터넷 연결
- 패키지 다운로드 및 네이버 맵 접속용

## 💡 Windows 전용 팁

### 팁 1: 바로가기 만들기
```
"실행.bat"를 마우스 오른쪽 클릭
→ "바로가기 만들기"
→ 바탕화면으로 이동
→ 이제 바탕화면에서 바로 실행!
```

### 팁 2: 작업 표시줄에 고정
```
"실행.bat" 바로가기를 작업 표시줄로 드래그
→ 클릭 한 번으로 실행!
```

### 팁 3: 시작 메뉴에 등록
```
폴더 전체를 복사:
C:\ProgramData\Microsoft\Windows\Start Menu\Programs\
→ 시작 메뉴에서 검색 가능!
```

### 팁 4: 예약 작업 설정
```
작업 스케줄러에서 자동 실행 설정 가능
→ 매일 자동으로 업데이트!
```

## 🐛 문제 해결 (간단 버전)

### 문제: Python을 찾을 수 없습니다
```
→ 설치.bat 실행 시 자동으로 다운로드 페이지 열림
→ 설치 후 PC 재부팅
```

### 문제: 패키지 설치 실패
```
→ 인터넷 연결 확인
→ 관리자 권한으로 실행
```

### 문제: Chrome 드라이버 오류
```
→ Chrome 업데이트
→ pip install --upgrade selenium
```

### 문제: 한글이 깨져요
```
→ 정상입니다! 프로그램은 정상 작동합니다.
→ 명령 프롬프트의 폰트 제한 때문입니다.
```
