## 💡 Windows 환경 특별 대응

### 1. 한글 경로 문제 해결
```python
# 배치 파일에 UTF-8 설정 추가
chcp 65001 > nul
```

### 2. 경로 구분자 자동 처리
```python
# Windows의 백슬래시(\) 자동 처리
os.path.join() 사용
```

### 3. 관리자 권한 필요 시 안내
```bat
# 권한 오류 발생 시 자동 안내
"관리자 권한으로 실행하세요"
```

### 4. Chrome 설치 경로 자동 감지
```bat
# 32비트/64비트 자동 감지
"%ProgramFiles%\Google\Chrome\"
"%ProgramFiles(x86)%\Google\Chrome\"
```

### 5. 폴더 자동 열기
```bat
# 완료 후 Explorer로 폴더 자동 열기
explorer downloads
```
