## 1. 프로그램 설치

### 1-1. Python 설치 (필수)

**다운로드:**
- https://www.python.org/downloads/
- **Python 3.12 이상** 버전 다운로드

**설치 시 주의사항:**
1. 설치 첫 화면에서 **"Add Python to PATH"** 체크박스를 **반드시 체크**하세요!
2. "Install Now" 클릭
3. 설치 완료 후 재부팅

**설치 확인:**
```cmd
python --version
```
결과: `Python 3.12.x` 또는 그 이상이 나오면 성공

### 1-2. Chrome 브라우저 (필수)

**다운로드:**
- https://www.google.com/chrome/

**설치:**
- 기본 설정으로 설치하면 됩니다.

## 2. 파일 다운로드

### 방법 1: GitHub에서 직접 다운로드

1. https://github.com/kingbong0612/naver-map-photo-downloader 접속
2. 녹색 "Code" 버튼 클릭
3. "Download ZIP" 클릭
4. 다운로드한 ZIP 파일을 원하는 위치에 압축 해제
   - 예: `C:\Users\사용자이름\Documents\naver-map-photo-downloader`

### 방법 2: Git으로 다운로드 (선택사항)

```cmd
cd C:\Users\사용자이름\Documents
git clone https://github.com/kingbong0612/naver-map-photo-downloader.git
```

## 3. 라이브러리 설치

### 3-1. 명령 프롬프트(cmd) 열기

**방법:**
1. 윈도우 키 + R
2. `cmd` 입력 후 엔터

### 3-2. 프로젝트 폴더로 이동

```cmd
cd C:\Users\사용자이름\Documents\naver-map-photo-downloader
```
*주의: 실제 압축 푼 경로로 변경하세요!*

### 3-3. 필수 라이브러리 자동 설치

```cmd
pip install -r requirements.txt
```

**설치되는 라이브러리:**
- `pandas` - 엑셀 파일 읽기/쓰기
- `openpyxl` - 엑셀 파일 처리
- `selenium` - 웹 브라우저 자동화
- `Pillow` - 이미지 처리
- `requests` - 이미지 다운로드

**설치 확인:**
```cmd
pip list
```
위의 라이브러리들이 목록에 나오면 성공!

## 4. 사용 방법

### 4-1. 엑셀 파일 준비

**필수 컬럼:**
- `지역` (예: 서울, 인천, 대전)
- `지역상세` (예: 중구, 강남구) - 없으면 비워두기
- `매장명` (예: 온아2호, 라힌)
- `네이버지도링크` (선택사항)

**예시:**
| 지역 | 지역상세 | 매장명 | 네이버지도링크 |
|------|---------|--------|---------------|
| 서울 | 중구 | 온아2호 | https://... |
| 인천 |  | 도투 | https://... |

---

### 4-2. 실행 방법

#### 방법 A: 배치 파일 실행 (가장 쉬움)

**업체 사진 다운로드:**
1. `실행_V4.bat` 더블클릭
2. 엑셀 파일 선택
   - `[1]` 입력 → 테스트용 (3개 매장)
   - `[2]` 입력 → 전체 (89개 매장)
3. 완료될 때까지 기다리기 (8-10분)

**네이버 플레이스 캡처:**
1. `캡처_플레이스.bat` 더블클릭
2. 엑셀 파일 선택
3. 완료될 때까지 기다리기 (5-8분)

**가격표 추출:**
1. `가격표추출.bat` 더블클릭
2. 엑셀 파일 선택
3. 완료될 때까지 기다리기 (5-8분)

#### 방법 B: 직접 명령어 실행

```cmd
# 업체 사진 다운로드
python download_naver_photos_v4.py 테스트_리스트.xlsx

# 네이버 플레이스 캡처
python capture_naver_place.py 테스트_리스트.xlsx

# 가격표 추출
python extract_price_table.py 테스트_리스트.xlsx
```

## 5. 문제 해결

### 문제 1: "python을 찾을 수 없습니다"

**해결:**
1. Python 설치 시 "Add Python to PATH" 체크 안 함
2. Python 재설치 필요 (체크박스 꼭 체크!)

---

### 문제 2: "pip를 찾을 수 없습니다"

**해결:**
```cmd
python -m ensurepip --upgrade
python -m pip install --upgrade pip
```

---

### 문제 3: Chrome 드라이버 오류

**해결:**
- Chrome 브라우저를 최신 버전으로 업데이트
- Chrome을 실행 중이면 종료 후 다시 시도

---

### 문제 4: 엑셀 파일을 읽을 수 없음

**해결:**
```cmd
pip install openpyxl --upgrade
```

---

### 문제 5: 이미지 다운로드 실패

**해결:**
1. 인터넷 연결 확인
2. 방화벽 설정 확인
3. 프로그램을 관리자 권한으로 실행

---

### 문제 6: 실패 목록 확인

**생성되는 파일:**
- `다운로드_실패_목록.txt` - 사진 다운로드 실패
- `캡처_실패_목록.txt` - 플레이스 캡처 실패
- `가격표추출_실패목록.txt` - 가격표 추출 실패

**내용:**
- 실패한 매장 목록
- 검색 URL
- 수동 처리 방법
