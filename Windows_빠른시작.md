# 🪟 Windows 빠른 시작 가이드

## 🎯 3단계로 끝내기 (초보자용)

### 1단계: 파일 준비 (1분)
1. 바탕화면에 `naver_map` 폴더 만들기
2. 다음 파일들을 폴더에 복사:
   - ✅ `설치.bat`
   - ✅ `실행.bat`
   - ✅ `naver_map_bulk_downloader.py`
   - ✅ `requirements.txt`
   - ✅ `리스트_네이버지도링크추가.xlsx`

### 2단계: 설치 (3분)
1. **`설치.bat`** 파일을 **더블클릭**
2. 자동으로 설치 진행
3. Python이 없으면 자동으로 다운로드 페이지 열림
4. Chrome이 없으면 자동으로 다운로드 페이지 열림

### 3단계: 실행 (20분)
1. **`실행.bat`** 파일을 **더블클릭**
2. 메뉴에서 선택:
   - `1` → 테스트 (3개 매장)
   - `2` → 전체 실행 (45개 매장)
3. 완료될 때까지 대기
4. `downloads` 폴더에서 결과 확인!

## ❓ 자주 묻는 질문 (FAQ)

### Q1: Python이 없는데 어떻게 하나요?
**A:** `설치.bat` 파일이 자동으로 다운로드 페이지를 열어줍니다.
      또는 https://www.python.org/downloads/ 에서 다운로드하세요.

### Q2: "Add Python to PATH"가 뭔가요?
**A:** Python 설치 시 첫 화면에 나오는 체크박스입니다.
      **반드시 체크**해야 합니다!

### Q3: Chrome이 없으면 어떻게 되나요?
**A:** Chrome이 필수입니다. `설치.bat`이 자동으로 다운로드 페이지를 열어줍니다.

### Q4: 중간에 멈춘 것 같아요
**A:** 괜찮습니다. 사진이 많으면 시간이 오래 걸립니다.
      검은 창에 진행률이 표시되는지 확인하세요.

### Q5: 중간에 멈추고 싶어요
**A:** **Ctrl + C** 키를 누르면 중단됩니다.
      이미 다운로드한 사진은 그대로 유지됩니다.

### Q6: 오류가 났어요!
**A:** 다음을 확인하세요:
   - ✅ Python 설치되어 있나요? → `python --version`
   - ✅ Chrome 설치되어 있나요?
   - ✅ 인터넷 연결되어 있나요?
   - ✅ 엑셀 파일이 폴더에 있나요?

### Q7: 다시 실행해도 되나요?
**A:** 네! 이미 다운로드한 사진은 건너뛰지 않고 덮어씁니다.
      필요하면 `downloads` 폴더를 백업하세요.

## 🛠️ 문제 해결

### ❌ "Python을 찾을 수 없습니다"
```
해결:
1. Python 설치 확인
2. 설치 시 "Add Python to PATH" 체크 확인
3. 명령 프롬프트 재시작
4. 안되면 PC 재부팅
```

### ❌ "파일을 찾을 수 없습니다"
```
해결:
1. 엑셀 파일이 같은 폴더에 있는지 확인
2. 파일 이름이 정확한지 확인
3. 파일 확장자(.xlsx)가 있는지 확인
```

### ❌ "권한이 없습니다"
```
해결:
1. 배치 파일을 마우스 오른쪽 클릭
2. "관리자 권한으로 실행" 선택
```

### ❌ "Chrome 드라이버 오류"
```
해결:
1. Chrome 브라우저 업데이트
2. 명령 프롬프트에서 실행:
   pip install --upgrade selenium
```
